Project 1

Live Chat Room

The Live Chat Room is a real-time chat application that utilizes HTML, JS, and CSS for the front end, and Firebase for the back end. With Google user authentication, it ensures secure and easy sign-in for users. Leveraging Firebase's real-time database, the app offers dynamic conversation experiences by providing live updates to users, creating a seamless and interactive communication platform.

Project 2

Tetris

Tetris is a fully interactive game built with React, TypeScript, and HTML. This project uses multiple custom React Hooks to manage state and side effects in functional components, resulting in a highly engaging and enjoyable gaming experience.

Project 3

Shoe Customizer

The Shoe Customizer is a application created using HTML, CSS, React-Three, and JavaScript. Through a Blender-rendered shoe model, users can dynamically customize different elements like mesh, stripes, laces, and sole, resulting in a highly personalized design experience. The incorporation of React-Three enables efficient manipulation of 3D objects, allowing users to interact fully with the shoe model through zooming, panning, and rotation features.

The Weather application is a dynamic and user-friendly tool developed using HTML, CSS, and JavaScript. By integrating the OpenWeatherMap API, the app fetches real-time weather data, ensuring users have access to accurate and up-to-date meteorological information.
